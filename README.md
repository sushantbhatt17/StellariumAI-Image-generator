# 🎨 AI Image Generation with Pollination AI (MERN Stack)

AI image generation is one of the most exciting advancements in artificial intelligence. With modern deep learning models, it is now possible to generate high-quality, realistic images from simple text prompts. These technologies are widely used in design, entertainment, marketing, and content creation.

This project leverages **Pollination AI API** to generate images from user input.

It is a **full-stack MERN application** that allows users to create, explore, and share AI-generated images with a community.

---

## ✨ Features

- 🎯 Modern and minimal UI for smooth user experience  
- 🎲 **"Surprise Me"** button for random prompt ideas  
- 🔍 Search functionality for exploring images  
- 📥 Download generated images  
- 🖱️ Hover effects showing prompt, username, and download option  
- 🌐 Community sharing feature

## ⚙️ How to Use

1. Visit the homepage to explore the **Community Showcase**
2. Use the search bar to find images by keyword or username
3. Click on **Create** to generate your own image
4. Enter:
   - Username  
   - Prompt  
   - OR click **Surprise Me**
5. Click **Generate**
6. Download or share your image with the community

---

## ⚙️ How It Works

- User submits a prompt from frontend (React)
- Backend (Node + Express) processes request
- Server calls **Pollination AI API**
- Generated image is returned
- Image is:
  - Stored in **MongoDB**
  - Uploaded to **Cloudinary**
- Frontend displays the image
- Users can share it in the community section
